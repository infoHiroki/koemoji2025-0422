# 音声会議インテリジェンスシステム要件定義書

## 2. 機能要件（続き）
### 2.2 ユーザー管理機能
1. **認証機能**
   * LINEアカウントでのログイン
   * 簡易ユーザー登録（メールアドレス）

2. **利用履歴管理**
   * 処理済み会議の一覧表示
   * 過去データへのアクセス・検索

### 2.3 課金・決済機能
1. **利用料金体系**
   * 従量課金プラン：8円/分（一般処理）、15円/分（プレミアム処理）
   * サブスクリプションプラン：月額1,500円（3時間/月）、月額4,000円（10時間/月）

2. **決済処理**
   * クレジットカード決済
   * LINE Pay連携
   * 利用明細の発行
